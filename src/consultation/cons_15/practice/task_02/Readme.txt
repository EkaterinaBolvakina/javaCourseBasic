Task No. 2
------------
Task: Weather data analyser

Goal: To develop a program for analysing weather data.

Description: You get a WeatherData object that contains information about
temperature in degrees Celsius and weather description
(e.g. "sunny", "cloudy", "rain", etc.).

You need to write a program that analyses this data and outputs
appropriate messages to the user.

Objectives:

    1. Check that the data is correct (the temperature should not be below -50 and above +50 degrees Celsius).
    2. Convert the weather description into an appropriate message:
        (a) "sunny" -> "Don't forget the sunscreen!"
        (b) "Cloudy" -> "You may need an umbrella."
        c) "rain" -> "Don't forget your umbrella!"
        (d) Any other description -> "Today's weather: [description]".

Requirements:

    Use the WeatherData class to represent weather data.
    Use functional interfaces such as Predicate to check temperature and
    Function to convert the weather description into a message.
    Output the resulting messages to the user.

Задание № 2
------------
Задача: Анализатор погодных данных

Цель: Разработать программу для анализа погодных данных.

Описание: Вы получаете объект WeatherData, который содержит информацию о
температуре в градусах Цельсия и описании погоды
(например, "солнечно", "облачно", "дождь" и т.д.).

Вам нужно написать программу, которая анализирует эти данные и выводит
соответствующие сообщения пользователю.

Задачи:

    1. Проверить корректность данных (температура не должна быть ниже -50 и выше +50 градусов Цельсия).
    2. Преобразовать описание погоды в соответствующее сообщение:
        a) "солнечно" -> "Не забудьте солнцезащитный крем!"
        b) "облачно" -> "Может быть, понадобится зонт."
        c) "дождь" -> "Не забудьте зонт!"
        d) любое другое описание -> "Сегодняшняя погода: [описание]".

Требования:

    Использовать класс WeatherData для представления данных о погоде.
    Использовать функциональные интерфейсы, такие как Predicate для проверки температуры и
    Function для преобразования описания погоды в сообщение.
    Вывести полученные сообщения пользователю.
