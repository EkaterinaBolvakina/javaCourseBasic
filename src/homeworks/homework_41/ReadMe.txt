## Homework

Task 1: Filtering and sorting a collection

Task: A list of strings is given.
You need to filter all strings from this list,
that begin with the letter "A" (case ignored), and
return the list of these strings sorted in alphabetical order.

Task 2: Grouping elements and counting the number of elements

Task: Given a list of Employee objects,
where each object contains the name and department where the employee works.
It is necessary to group employees by departments and calculate the number of employees in each department.
number of employees in each department.

Task 3: Object List Conversion and Aggregation

Task: There is a list of order objects (Order), where each order
contains a list of purchases (List<Purchase>) and the date of the order.
Each purchase contains the name of the product and its cost.
You need to calculate the total cost of all purchases,
made in the current month.

## Домашнее задание

Задача 1: Фильтрация и сортировка коллекции

Задание: Дан список строк.
Необходимо отфильтровать из этого списка все строки,
которые начинаются на букву "A" (без учёта регистра), и
вернуть список этих строк, отсортированный в алфавитном порядке.

Задача 2: Группировка элементов и подсчёт количества

Задание: Дан список объектов сотрудников (Employee),
где каждый объект содержит имя и отдел, в котором работает сотрудник.
Необходимо сгруппировать сотрудников по отделам и подсчитать
количество сотрудников в каждом отделе.

Задача 3: Преобразование списка объектов и агрегация

Задание: Дан список объектов заказов (Order), где каждый заказ
содержит список покупок (List<Purchase>) и дату заказа.
Каждая покупка содержит название товара и его стоимость.
Необходимо рассчитать общую сумму стоимости всех покупок,
сделанных в текущем месяце.
