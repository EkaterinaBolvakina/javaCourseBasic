------- ENG -------
Option 2. Car showroom management system with model evaluations

Aim: To develop a system for managing cars in a car showroom, including a model evaluation function.

Objectives:

    Create a Car class with information about the make, model, year, price and rating of the car.
    Create a CarDealer class that uses HashMap to store cars, where the key is a unique car identifier (e.g. VIN number) and the value is a Car object.
    In the CarDealer class, implement methods to add new cars, remove cars from the list, update information about a particular car (e.g., price or rating),
    and search for cars by various criteria (make, model, year of manufacture).
    Add functionality for buyers to rate cars and sort cars by rating, price or year of manufacture.
    Implement the ability to display information about all cars in stock or cars of a specific brand.

------- DEU -------
Option 2. Autohausverwaltungssystem mit Modellbewertungen

Ziel: Entwicklung eines Systems zur Verwaltung von Autos in einem Autohaus, einschließlich einer Modellbewertungsfunktion.

Zielsetzungen:

    Erstellen einer Klasse Car mit Informationen über Marke, Modell, Baujahr, Preis und Bewertung des Fahrzeugs.
    Erstellen Sie eine Klasse CarDealer, die HashMap verwendet, um Autos zu speichern,
    wobei der Schlüssel eine eindeutige Fahrzeugkennung (z.B. VIN-Nummer) und der Wert ein Car-Objekt ist.
    Implementieren Sie in der Klasse CarDealer Methoden zum Hinzufügen neuer Autos, zum Entfernen von Autos aus der Liste,
    zum Aktualisieren von Informationen über ein bestimmtes Auto (z. B. Preis oder Bewertung) und
    zum Suchen nach Autos anhand verschiedener Kriterien (Marke, Modell, Baujahr).
    Hinzufügen von Funktionen, mit denen Käufer Autos bewerten und nach Bewertung, Preis oder Baujahr sortieren können.
    Implementierung der Möglichkeit, Informationen über alle auf Lager befindlichen Fahrzeuge oder Fahrzeuge einer bestimmten Marke anzuzeigen.

------- RUS -------
Вариант 2. Система управления автомобильным салоном с оценками моделей

Цель: Разработать систему для управления автомобилями в автосалоне, включая функцию оценки моделей.

Задачи:

    Создать класс Car с информацией о марке, модели, годе выпуска, цене и рейтинге автомобиля.
    Создать класс CarDealer, который использует HashMap для хранения автомобилей,
    где ключом является уникальный идентификатор автомобиля (например, VIN-номер), а значением - объект Car.
    В классе CarDealer реализовать методы для добавления новых автомобилей, удаления автомобилей из списка,
    обновления информации о конкретной машине (например, цены или рейтинга),
    а также поиска автомобилей по различным критериям (марка, модель, год выпуска).
    Добавить функционал для оценки автомобилей покупателями и сортировки автомобилей по рейтингу, цене или году выпуска.
    Реализовать возможность вывода информации о всех автомобилях в наличии или автомобилях определенной марки.
